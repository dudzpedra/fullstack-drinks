# Drinks Review App
A simple app to explore cocktails from some local bars!

## Stack
### Backend
- Node.js
- Express.js
- MongoDB
- Cors
- dotenv
- nodemon

### Frontend
- React
- Redux Toolkit
- Axios 
- react-router-dom
- styled-components
- polished library
